Resumo da Aula 3 – Fundamentos de JavaScript (26/03/2026)

Nesta aula, aprendemos como interagir com o usuário usando JavaScript. Utilizamos prompt para receber dados (como nome, idade e números) e alert para mostrar mensagens na tela. Também vimos como converter valores digitados em números com parseInt para poder fazer cálculos.

Por fim, foi feito um exemplo prático somando dois números informados pelo usuário e exibindo o resultado tanto na tela (alert) quanto no console (console.log).
